📊 Scoring System

Every term is scored on:
• Base Definition - 20 points for the essentials
• Humor - Up to 30 points for making us laugh!
• Explanation - 20 points for context
• Cross-references - Up to 20 points for connections
• Tags - 10 points for categorization

💯 Score 90+ to become a legend!
